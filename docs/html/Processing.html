ChromProcess.Processing.chromatogram.background_subtraction.ic_background_subtraction(chromatogram, threshold=500)¶

Gets the ion chromatograms of the analysis and reconstitutes the total ion chromatogram ommitting m/z signals which do not exceed a threshold.

Parameters

chromatogram: ChromProcess Chromatogram object.

Chromatogram to be processed.

threshold: float

Ion chromatograms which do not exceed this threshold will be removed before the total ion chromatogram is reconsituted.

Returns

1D numpy array.

Original signal if not mass spectra information present. processed signal if ms info present.
